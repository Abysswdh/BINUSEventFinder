## Legenda Simbol

| Simbol | Nama | Deskripsi |
|--------|------|-----------|
| ● | Initial Node | Titik awal aktivitas |
| ◉ | Final Node | Titik akhir aktivitas |
| ┌───────┐ | Activity | Aktivitas/aksi yang dilakukan |
| ◇ | Decision | Titik keputusan/percabangan |
| ───────► | Flow | Arah alur aktivitas |
| │ | Swimlane | Pembatas antar aktor |
